Question-1: What are the different ways to select an element in the DOM?
Using methods like getElementById, getElementsByClassName, querySelector, or querySelectorAll to select elements in the DOM.
Question-2: What is the difference between innerHTML, innerText, and textContent ?
innerHTML includes HTML, innerText shows visible text, and textContent displays all text, including hidden content.
Question-3: What is event delegation in the DOM?
Event delegation is adding one event listener to a parent element to manage events for its children.
Question-4: What is event bubbling in the DOM?
Event bubbling is when an event on a child element moves up through its parent elements in the DOM.
Question-5: How do you create, add, and remove elements using JavaScript?
Create elements with createElement, add them using appendChild or insertBefore, and remove them with removeChild or remove.
